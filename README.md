## 课程亮点

1. 编码语言：TypeScript
2. 使用框架：Vue3
3. 代码风格：组合式
4. 构建工具：Vite
5. 路由管理：vue-router
6. 集中式状态：Pinia状态管理
7. UI组件库：Element-UI
8. 网络请求：Axios
